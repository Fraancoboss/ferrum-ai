# Providers Governance Check

Use this when `Providers` looks wrong or when you want a fast health pass.

## 1. Open Governance

Go to:

```text
Providers > Governance
```

## 2. Check the summary row

Confirm:

- `Ollama mode`
- `Authority`
- `Inventory issues`
- `Recent jobs`

## 3. Inspect local providers

For `Ollama` and `llama.cpp`, check:

- status label
- version
- detail text
- issue pills

## 4. Refresh diagnostics

Use:

- `Refresh all`

Then confirm:

- endpoint still matches expected `OLLAMA_API_BASE`
- job states are current
- issues are either gone or clearly explained
